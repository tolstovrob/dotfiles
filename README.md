# tolstovrob/dotfiles

Мои конфиги для zsh, vim, ghostty. Включая темы, алиасы, шорткаты и прочее

## Как установить?

Удобно использовать утилиту [just](https://github.com/casey/just):

```bash
git clone git@github.com:tolstovrob/dotfiles.git
cd dotfiles
just link
```

Перезапустите оболочку и терминал для лучшего эффекта
